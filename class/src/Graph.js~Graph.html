    public * iterEdges(options: Object) source

    Iterates over the edges in the graph. If the edges parameter is specified only those edges are considered. Furthermore if a filter function is supplied only edges that pass the test are included. It is also possible to apply a function to an edge and yield the result instead.

    Params:

    NameTypeAttributeDescription
    options Object
    • optional
    • default: {}

    Used for various options.
    options.filter Function
    • optional

    The filter function to apply. It gets an edge as the first parameter and this graph as the second.
    options.map Function
    • optional

    The function to use for mapping. It gets a node as the first parameter and this graph as the second.
    options.edges String | Iterator<String> | Edge | Iterator<Edge>
    • optional

    Restricts the edges to iterate over. Specifying IDs is sufficient.
