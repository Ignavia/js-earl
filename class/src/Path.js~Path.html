    public isCycle: Boolean source

    If this path is a cycle. This means the first and last node are the same and the length of the path is at least 1.

    public isSimpleCycle: Boolean source

    If the path is a simple cycle. This means it is a cycle and no node except the first and last node is visited twice.

    public isSimplePath: Boolean source

    If the path is simple. This means no node is visited twice.
