    public getNumberOfAdjacentNodes(direction: String): Number source

    Returns the number of adjacent nodes. The result depends on the given direction.

    Params:

    NameTypeAttributeDescription
    direction String
    • optional
    • default: "all"
    • "all": All adjacent nodes are considered.
    • "out": Only nodes that are the target node of at least one edge starting at this node are selected.
    • "inc": Only nodes that are the source node of at least one edge ending at this node are chosen.

    Return:

    Number

    The number of adjacent nodes.

    Params:

    NameTypeAttributeDescription
    node String | Node

    The other ending point of the edges. Specifying an ID is enough.

    • optional

    Return:

    Number

    The number of edges between this node and the given node.
