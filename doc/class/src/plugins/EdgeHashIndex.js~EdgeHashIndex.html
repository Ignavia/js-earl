    EdgeHashIndex

    Extends:

    @ignavia/util~Plugin → HashIndex → EdgeHashIndex

    An index that maps from a hash value to the IDs of all edges with that hash value. It adds a getter and an iterator to a graph when it is plugged in.

    mixinNames Object

    Provides names for the mixin functions.
    mixinNames.getterName String
    • optional

    The name of the getter function. This returns the IDs of all edges with the given hash. It is also possible to provide an edge object instead of a hash and set the second parameter to true. Then the edge is hashed first and the result is used as hash.
    mixinNames.iteratorName String
    • optional

    The name of the iterator function. This iterates over the IDs of all edges with the given hash. It is also possible to provide an edge object instead of a hash and set the second parameter to true. Then the edge is hashed first and the result is used as hash.
    mixinNames.counterName String
    • optional

    The name of the counter function. This returns the number of all edges with the given hash. It is also possible to provide an edge object instead of a hash and set the second parameter to true. Then the edge is hashed first and the result is used as hash.
    mixinNames.testerName String
    • optional

    The name of the tester function. This tests whether edges with the given hash value exist. It is also possible to provide an edge object instead of a hash and set the second parameter to true. Then the edge is hashed first and the result is used as hash.
